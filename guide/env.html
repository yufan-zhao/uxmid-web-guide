# 开发环境搭建

  1. 准备一个gitee (opens new window)账号。

    • 公司采用的是gitee企业版，可以使用已有帐号或者自行注册一个个人账号，等待管理员添加成员

  2. 下载Visual Studio Code (opens new window)和Source Tree (opens new window)。

    • VS code插件推荐 (opens new window)
    • Source Tree是作为git仓库可视化操作的客户端，如有其他工具都可替代

  3. 下载并安装Node (opens new window)

    • 请全局安装yarn (opens new window)，公司前端项目采用yarn替代npm
    • 替换npm仓库地址为淘宝镜像
    npm config set registry https://registry.npm.taobao.org
    • 安装Electron等特殊包操作说明 (opens new window)
    ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/ npm install -g electron

  4. 注册蓝湖 (opens new window)账号

    • 公司项目的原型图和设计图都会提交蓝湖，注册之后等待管理员邀请

  5. 相关工具、文档推荐

    • 科学上网 (opens new window)
    • 截图贴图工具Snipaste (opens new window)
    • Typescript中文手册 (opens new window)
    • Flex布局完全指南 (opens new window)
    • Vue源码学习 (opens new window)
    • View UI官方文档 (opens new window)
